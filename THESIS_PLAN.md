## 4. Quellen-Management

**Wichtig:** Prof. Binder verlangt **deutsche Fußnoten-Zitierweise** (Kurztitel oder Jahreszahl), *nicht* Harvard im Text.

- Minimum **30 Quellen**; Mix ca. **50 % Standardlehrbücher / 50 % aktuelle Fachliteratur** (Zeitschriften, Zeitungen, Internet).
- Immer aktuellste Primärquelle (neueste Auflage/Jahrgang); Internetquellen mit vollständigem Link.
- Fußnote im Text: `Meisenbach C. (1999), S. 14` — bei Wiederholung: `ebenda` bzw. `ebenda S. 12`.
- Wörtliche Zitate in „…", keine Textänderung; sinngemäße Zitate: Änderung zulässig; lange Übernahmen vermeiden.
- Literaturverzeichnis-Formate (Buch, Sammelband, Dissertation, Zeitschrift, Zeitung, o.V./o.J./o.O.): siehe `00_Expose/Vorgaben_Prof_Binder_Zitierrichtlinien.pdf`, Punkt 7. Mehrere Werke eines Autors im selben Jahr: a, b, c.
- Quellenliste wird laufend gepflegt: `01_Literatur/Quellenliste.md` (anlegen in P1).
- Empfohlene Grundlagenliteratur zum wiss. Arbeiten: Kornmeier, Theisen.

## 5. Schreibstil-Regeln (bei jedem Entwurf prüfen)

Hochdeutsch, keine zu langen Sätze, kein Ich/Wir, keine Übertreibungen („hohe" statt „gigantische" Kosten), keine Füllwörter und Floskeln, kein journalistischer Stichwort-Stil, vollständige konkrete Sätze idealerweise mit Quellangabe und eigenem Praxisbeispiel bzw. abgeleiteter Handlungsempfehlung. Erkennbare inhaltliche Eigenleistung. Natürlicher, menschlicher wissenschaftlicher Duktus — keine generisch klingenden Formulierungen.

## 6. Word-Formatvorlage (Setup in P1)

| Element | Vorgabe |
|---|---|
| Schrift | Arial, 12 pt, Blocksatz, Zeilenabstand 1,5 |
| Ränder | oben 2,5 / unten 2,0 / links 4,0 / rechts 2,0 cm |
| Überschriften | fett, linksbündig, nicht unterstrichen; max. 4 Gliederungsebenen; kein Punkt nach letzter Ziffer (2.2.1); jede Ebene ≥ 2 Punkte |
| Fußnoten | unten auf der Seite, 8–10 pt, nummeriert |
| Seitenzahlen | Verzeichnisse/Anhang/Literatur/Erklärung: römisch · Textteil: arabisch |
| Reihenfolge | Titelblatt (o.S.) → Inhaltsverz. → Abkürzungsverz. → Abbildungsverz. → Text → Anhang → Literaturverz. → Erklärung (handschriftl. Unterschrift); Eidesstattliche Erklärung zu Beginn |
| Abbildungen | keine 3D; alle im Abbildungsverzeichnis; im Text erklären; Tabellen ggf. eigenes Verzeichnis |
